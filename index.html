font-sizes

The em value and relativity

The value is relative to the parent's font-size.
As a result, the value will cascade if used on child elements.

Parent container: 20px;

font-size: 1.2em; (24px)
font-size: 1.2em; (28.8px)
font-size: 1.2em; (34.56px)

Other values

  • Rem value is relative to the root element's font-size, which is the html element.
